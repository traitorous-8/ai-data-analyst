# ai-data-analyst
Интерактивное веб-приложение для анализа табличных данных (CSV/Excel) с использованием Large Language Model (OpenAI API).

Проект позволяет загружать датасеты, строить визуализации и получать осмысленный текстовый анализ данных по пользовательскому запросу.

Возможности:

- Загрузка CSV и Excel файлов
- Просмотр и первичный анализ данных
- Текстовый анализ данных с помощью LLM
- Построение графиков (столбчатые, линейные, круговые и автоматический выбор типа графика)
- Удобный интерфейс на Streamlit

Используемые технологии:

- Python 3.9+
- Streamlit
- Pandas 
- Matplotlib
- OpenAI API
- python-dotenv
